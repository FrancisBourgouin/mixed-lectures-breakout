# Todoinator

Todo! We should be able to add a task, due date, color code, mark them as complete, delete, (edit stretch)

## Data

### Todo

```jsx
const todo = {
  id: "",
  task: "",
  dueDate: "",
  color: "",
  isComplete: false,
};
```

### Todo

Immutability

```jsx
const todos = { 1: todo, 2: todo };

const todos = [];
```
